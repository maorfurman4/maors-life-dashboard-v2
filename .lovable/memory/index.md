# Project Memory

## Core
"החיים של מאור" — personal life management app, Hebrew RTL, single user (מאור).
Dark premium design. Mobile-first. Font: Heebo.
Module accents: sport=neon green, nutrition=turquoise, finance=gold, market=trading green/red, work=blue.
Layout: desktop sidebar right (RTL), mobile bottom nav. TanStack Start + Tailwind v4.
User has shoulder sensitivity — sport features filter shoulder-safe exercises by default.
